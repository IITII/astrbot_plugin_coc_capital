# 📦 都城突袭奖章查询插件

基于 AstrBot 实现，支持多部落批量查询、并发限速、进攻奖励计算与表格对齐输出。  

---

## 📌 功能

- 🏹 同时查询多个部落的周末突袭奖励  
- 🔢 自动计算进攻奖励（按区块星数 + 奖章规则）  
- 📉 并发限速控制，避免 API 封禁  
- 📊 表格对齐输出（适合群聊展示）  
- 💾 缓存 clan 和 raid 数据（减少重复请求）  
- 🧠 支持插件配置控制行为
